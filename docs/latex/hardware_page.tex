\end{DoxyItemize}\hypertarget{hardware_page_common_problems}{}\doxysection{\texorpdfstring{Common Challenges}{Common Challenges}}\label{hardware_page_common_problems}
This section shows some common challenges we encountered and how to troubleshoot them\+:


\begin{DoxyItemize}
\item {\bfseries{Problem\+:}} Solder sticking to the board causing parts to short.
\begin{DoxyItemize}
\item {\bfseries{Solution\+:}} Look carefully for the soldering blob, do not use too much soldering wire; blobs are caused by excess solder.
\end{DoxyItemize}
\item {\bfseries{Problem\+:}} The library does not show up in the Fusion 360 e\+Drawings.
\begin{DoxyItemize}
\item {\bfseries{Solution\+:}}
\begin{DoxyEnumerate}
\item Download the part with the STEP and the library from Digi\+Key.
\item Open Fusion 360.
\item Create a new folder and a new electronics library.
\item Open Library Manager and import libraries from local disk.
\item Right-\/click the library and click "{}\+Edit Fusion 360 copy"{}.
\item Click Footprint and move the name and values.
\item Right-\/click and select 3D sketch.
\item Open the STEP file and drag it into the 3D drawing; it should be linked after you save.
\end{DoxyEnumerate}
\end{DoxyItemize}
\item {\bfseries{Problem\+:}} The limit switch does not trigger the GPIO.
\begin{DoxyItemize}
\item {\bfseries{Solution\+:}} This can be due to the LED in the limit switch that is included in the product; it reduces the voltage output. It is recommended to short the LED and use the limit switch.
\end{DoxyItemize}
\item {\bfseries{Problem\+:}} In Fusion 360, routing error "{}\+PCB out of Sync"{} which doesn\textquotesingle{}t let you route after.
\begin{DoxyItemize}
\item {\bfseries{Solution\+:}} Push the schematic, then the footprint so it updates in the PCB routing.
\end{DoxyItemize}
\item {\bfseries{Problem\+:}} In Fusion 360, routing error "{}\+Overlap"{} in the microcontroller ground vias.
\begin{DoxyItemize}
\item {\bfseries{Solution\+:}} Just click on the part and select "{}\+Approved"{} in the DRC.
\end{DoxyItemize}
\item {\bfseries{Problem\+:}} How do you know how much trace width you need?
\begin{DoxyItemize}
\item {\bfseries{Solution\+:}} Use a trace width calculator. I recommend \href{https://www.4pcb.com/trace-width-calculator.html}{\texttt{ this one}}.
\end{DoxyItemize}
\item {\bfseries{Problem\+:}} How do you know if the LED or parts are connected on your PCB correctly so it doesn\textquotesingle{}t accidentally short circuit?
\begin{DoxyItemize}
\item {\bfseries{Solution\+:}} Use a multimeter and check for continuity; if it is connected correctly, it should beep. 
\end{DoxyItemize}
\end{DoxyItemize}
